# Prime Empire Guild Website 🎮⚔️

Welcome to the **Prime Empire** Guild Management Website! An epic, fully-featured guild website with player management capabilities.

## 🎯 Features

✅ **Home Page** - Stunning hero section with guild stats  
✅ **About Us** - Guild mission, values, and history  
✅ **Players Directory** - Browse all guild members  
✅ **Player Profiles** - Click any player to see full details  
✅ **Admin Dashboard** - Manage players (add, edit, delete)  
✅ **Secure Login** - Password-protected admin access  
✅ **Responsive Design** - Works on all devices  
✅ **No Backend Needed** - All data stored locally in your browser  

## 🚀 Quick Start

1. **Open `index.html`** in any web browser
2. **Navigate to Admin** section (click 🔐 Admin button)
3. **Login** with default password: `admin123`
4. **Change your admin password immediately** for security
5. **Start adding guild members!**

## 👥 Managing Players

### Add a Player:
1. Go to **Admin** section
2. Fill in player details:
   - **Name** (required)
   - **Role** (Warrior, Mage, Archer, Rogue, etc.)
   - **Level** (character level)
   - **Class** (character class)
   - **Join Date** (when they joined the guild)
   - **Bio/Description** (player bio or achievements)
3. Click "Add Player"

### Edit a Player:
1. In Admin panel, find the player
2. Click "Edit"
3. Update the information
4. Confirm changes

### Delete a Player:
1. Click "Delete" next to the player
2. Confirm deletion

## 🔧 Troubleshooting

**Lost your password?**
- Clear your browser's localStorage and start fresh

**Data not showing?**
- Check browser console (F12) for errors
- Make sure JavaScript is enabled
- Clear browser cache if needed

**Players not saving?**
- Browser must support localStorage
- Check if localStorage is not full
- Try a different browser
